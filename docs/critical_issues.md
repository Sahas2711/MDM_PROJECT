**Impact**: The page claims to be an "AI Crop Recommendation Engine" but contains zero AI. It is a UI mockup presented as a functional feature.

### Issue 7: DuckDuckGo Price API Unreliable
**Confidence: High**
**Location**: `backend/market_data.py`

The `fetch_market_prices()` function queries the DuckDuckGo Instant Answer API for live crop prices. This API:
- Is not an official API (no SLA, no documentation, subject to change)
- Rarely returns structured price data in Abstract/Answer fields
- Has a 4-second timeout

In practice, the fallback hardcoded prices (`_FALLBACK_PRICES`) are almost always used. The `price_source` field in the response will show `"dataset_median"` for virtually every request, not `"web_search"`.

**Impact**: The "live web price search" feature is effectively non-functional. The SmartDecision pipeline always uses hardcoded fallback prices.

### Issue 13: KMeans Float32 Dtype Fragility
**Confidence: High**
**Location**: `backend/predict.py`, line: `scaled = scaler.transform(features).astype(np.float32)`

The code explicitly documents this as a known bug fix: StandardScaler outputs float64, KMeans C extension requires float32. The fix is applied, but the comment in `smart_decision.py` node 6 "why" field still says "If this node still shows FAILED, a different runtime issue is present" — indicating this was a real production failure that was patched.

**Impact**: If the astype() cast is ever removed or the sklearn version changes behavior, KMeans will silently fail.

### Issue 14: YOLO Only Detects 4 Fruit Types
**Confidence: High**
**Location**: `backend/predict_image.py`, `YOLO_ALLOWED_LABELS = {"apple", "banana", "orange", "carrot"}`

YOLOv8 is trained on COCO (80 classes). Only 4 are allowed: apple, banana, orange, carrot. The fruit_classifier supports 10 fruit types (apple, banana, orange, mango, strawberry, potato, cucumber, carrot, tomato, bell pepper). Mango, strawberry, potato, cucumber, tomato, and bell pepper cannot be detected by YOLO — the pipeline will return `fruit_detected=False` for these crops even if the image clearly shows them.

**Impact**: The system fails silently for 6 of the 10 supported fruit types. A mango image will always return "NOT FRUIT IMAGE".
